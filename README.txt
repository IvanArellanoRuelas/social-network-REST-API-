This project consists of the development of a REST API for the fictitious social network Like Me, divided into two parts.

Part 1
The main focus is the creation of posts and interaction through likes. As a full stack developer, the challenge includes

Implement GET and POST routes on an Express server.
Connect to a PostgreSQL database to retrieve and store messages, using the pg package.
Enable CORS and perform basic SQL queries to handle messages.

Part 2
In the second part, the functionalities are expanded by allowing the modification and deletion of posts, along with error handling

Implement PUT and DELETE routes to modify and delete records in the database.
Manage possible errors with try-catch in SQL queries.
This project demonstrates skills in creating a REST API with database interaction and HTTP route management.
